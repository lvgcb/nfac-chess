# ♟️ NFAC Chess

Шахматная платформа с ИИ-тренером, мультиплеером и внутриигровой экономикой. Построено на TanStack Start и Lovable Cloud.

🔗 **Демо:** https://nfac-chess.lovable.app

---

## ✨ Возможности

### 🎮 Игра против ИИ
- Несколько уровней сложности (выбор сверху страницы)
- Центрированная доска, удобный интерфейс
- Светлая (бежевая) и тёмная темы

### 🧠 ИИ-тренер (анализ партий)
- После каждой партии открывается окно: **Новая игра** или **Анализ от тренера**
- Тренер разбирает **каждый ход**, объясняет его и предлагает лучший вариант, если он есть
- Ключевые моменты партии (брillиантные ходы, зевки) выделяются отдельно
- Интерактивная **доска для исследования** — можно пробовать свои варианты прямо в анализе

### 🌐 Мультиплеер
- Очередь матчмейкинга — система автоматически находит соперника
- Игра в реальном времени через Supabase Realtime
- Победитель получает **+10 монет**, проигравший теряет **−10 монет**
- По окончании матча — окно с кнопками **Играть снова** и **Анализ тренера**

### 👤 Аккаунт и профиль
- Регистрация и вход по email/паролю
- Личный кабинет с балансом монет и историей транзакций
- Симулированная покупка пакетов монет

### 💰 Внутриигровая валюта и магазин
- Монеты начисляются за победы и ничьи
- Магазин с реальными призами: турнирные билеты, шахматные часы, наборы фигур, ноутбук и др.
- Покупка списывает монеты атомарно через RPC `purchase_shop_item`

### 🍔 Навигация
- Всё меню (Игра, Мультиплеер, Магазин, Профиль, Тема, Вход/Выход) свёрнуто в один аккуратный hamburger-выпадающий список

| Слой | Технология |
|---|---|
| Фреймворк | TanStack Start v1 (React 19, Vite 7) |
| Стили | Tailwind CSS v4 + дизайн-токены в `src/styles.css` |
| Бэкенд | Lovable Cloud (Supabase: Postgres + Auth + Realtime) |
| Шахматный движок | `chess.js` |
| ИИ | Lovable AI Gateway (Google Gemini) для анализа партий |
| Деплой | Cloudflare Workers (через Lovable) |

## 🗄 База данных

Основные таблицы:
- `profiles` — пользователи и баланс монет
- `coin_transactions` — история операций с монетами
- `shop_items` — каталог магазина
- `orders` — оформленные заказы
- `matchmaking_queue` — очередь поиска соперника
- `matches` — активные и завершённые партии (с историей ходов)

Ключевые RPC-функции:
- `award_coins(amount, reason, metadata)` — начисление монет
- `purchase_shop_item(item_id)` — атомарная покупка
- `join_matchmaking()` — вход в очередь / создание матча
- `make_match_move(match_id, fen, san)` — валидация хода и обновление FEN
- `finish_match(match_id, result)` — завершение партии и расчёт монет

Все таблицы защищены **Row Level Security**.

```bash
bun install

```

## 🎯 Игровой цикл

1. Зарегистрируйся → получи 100 стартовых монет
2. Сыграй против ИИ или против живого соперника
3. Победил в мультиплеере? +10 монет 🪙
4. Открой анализ тренера и разбери свою партию
5. Накопи монеты и выкупи приз в магазине
